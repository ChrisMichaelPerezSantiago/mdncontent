## ✔️ hanging-punctuation 
 ---<br/>title: hanging-punctuation<br/>slug: Web/CSS/hanging-punctuation<br/>tags:<br/>  - CSS<br/>  - CSS Property<br/>  - CSS Text<br/>  - Reference<br/>  - recipe:css-property<br/>browser-compat: css.properties.hanging-punctuation<br/>---<br/>{{CSSRef}}<br/><br/>The **`hanging-punctuation`** [CSS](/en-US/docs/Web/CSS) property specifies whether a punctuation mark should hang at the start or end of a line of text. Hanging punctuation may be placed outside the line box.<br/><br/>```css<br/>/* Keyword values */<br/>hanging-punctuation: none;<br/>hanging-punctuation: first;<br/>hanging-punctuation: last;<br/>hanging-punctuation: force-end;<br/>hanging-punctuation: allow-end;<br/><br/>/* Two keywords */<br/>hanging-punctuation: first force-end;<br/>hanging-punctuation: first allow-end;<br/>hanging-punctuation: first last;<br/>hanging-punctuation: last force-end;<br/>hanging-punctuation: last allow-end;<br/><br/>/* Three keywords */<br/>hanging-punctuation: first force-end last;<br/>hanging-punctuation: first allow-end last;<br/><br/>/* Global values */<br/>hanging-punctuation: inherit;<br/>hanging-punctuation: initial;<br/>hanging-punctuation: revert;<br/>hanging-punctuation: revert-layer;<br/>hanging-punctuation: unset;<br/>```<br/><br/>## Syntax<br/><br/>The `hanging-punctuation` property may be specified with one, two, or three space-separated values.<br/><br/>- **One-value** syntax uses any one of the keyword values in the list below.<br/>- **Two-value** syntax uses one of the following:<br/><br/>  - `first` together with any one of `last`, `allow-end`, or `force-end`<br/>  - `last` together with any one of `first`, `allow-end`, or `force-end`<br/><br/>- **Three-value** syntax uses one of the following:<br/><br/>  - `first`, `allow-end`, and `last`<br/>  - `first`, `force-end`, and `last`<br/><br/>### Values<br/><br/>- `none`<br/>  - : No character hangs.<br/>- `first`<br/>  - : An opening bracket or quote at the start of the first formatted line of an element hangs.<br/>- `last`<br/>  - : A closing bracket or quote at the end of the last formatted line of an element hangs.<br/>- `force-end`<br/>  - : A stop or comma at the end of a line hangs.<br/>- `allow-end`<br/>  - : A stop or comma at the end of a line hangs if it does not otherwise fit prior to justification.<br/><br/>## Formal definition<br/><br/>{{cssinfo}}<br/><br/>## Formal syntax<br/><br/>{{csssyntax}}<br/><br/>## Examples<br/><br/>### Setting opening and closing quotes to hang<br/><br/>#### HTML<br/><br/>```html<br/><p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Curabitur dignissim nunc mauris, et sollicitudin est scelerisque sed. Praesent laoreet tortor massa, sit amet vulputate nulla pharetra ut.</p><br/>```<br/><br/>#### CSS<br/><br/>```css<br/>p {<br/>  hanging-punctuation: first last;<br/>  margin: .5rem;<br/>}<br/>```<br/><br/>#### Result<br/><br/>{{EmbedLiveSample(Setting_opening_and_closing_quotes_to_hang)}}<br/><br/>## Specifications<br/><br/>{{Specifications}}<br/><br/>## Browser compatibility<br/><br/>{{Compat}}<br/><br/>## See also<br/><br/>- {{cssxref('text-indent')}}<br/>- [CSS Tricks: Hanging punctuation](https://css-tricks.com/almanac/properties/h/hanging-punctuation/)<br/>
