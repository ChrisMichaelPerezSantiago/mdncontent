## ✔️ border-radius 
 ---<br/>title: border-radius<br/>slug: Web/CSS/border-radius<br/>tags:<br/>  - CSS<br/>  - CSS Borders<br/>  - CSS Property<br/>  - Reference<br/>  - recipe:css-shorthand-property<br/>browser-compat: css.properties.border-radius<br/>---<br/>{{CSSRef}}<br/><br/>The **`border-radius`** [CSS](/en-US/docs/Web/CSS) property rounds the corners of an element's outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners.<br/><br/>{{EmbedInteractiveExample(pages/css/border-radius.html)}}<br/><br/>The radius applies to the whole {{cssxref(background)}}, even if the element has no border; the exact position of the clipping is defined by the {{cssxref(background-clip)}} property.<br/><br/>The `border-radius` property does not apply to table elements when {{cssxref(border-collapse)}} is `collapse`.<br/><br/>> **Note:** As with any shorthand property, individual sub-properties cannot inherit, such as in `border-radius:0 0 inherit inherit`, which would partially override existing definitions. Instead, the individual longhand properties have to be used.<br/><br/>## Constituent properties<br/><br/>This property is a shorthand for the following CSS properties:<br/><br/>- [`border-top-left-radius`](/en-US/docs/Web/CSS/border-top-left-radius)<br/>- [`border-top-right-radius`](/en-US/docs/Web/CSS/border-top-right-radius)<br/>- [`border-bottom-right-radius`](/en-US/docs/Web/CSS/border-bottom-right-radius)<br/>- [`border-bottom-left-radius`](/en-US/docs/Web/CSS/border-bottom-left-radius)<br/><br/>## Syntax<br/><br/>```css<br/>/* The syntax of the first radius allows one to four values */<br/>/* Radius is set for all 4 sides */<br/>border-radius: 10px;<br/><br/>/* top-left-and-bottom-right | top-right-and-bottom-left */<br/>border-radius: 10px 5%;<br/><br/>/* top-left | top-right-and-bottom-left | bottom-right */<br/>border-radius: 2px 4px 2px;<br/><br/>/* top-left | top-right | bottom-right | bottom-left */<br/>border-radius: 1px 0 3px 4px;<br/><br/>/* The syntax of the second radius allows one to four values */<br/>/* (first radius values) / radius */<br/>border-radius: 10px / 20px;<br/><br/>/* (first radius values) / top-left-and-bottom-right | top-right-and-bottom-left */<br/>border-radius: 10px 5% / 20px 30px;<br/><br/>/* (first radius values) / top-left | top-right-and-bottom-left | bottom-right */<br/>border-radius: 10px 5px 2em / 20px 25px 30%;<br/><br/>/* (first radius values) / top-left | top-right | bottom-right | bottom-left */<br/>border-radius: 10px 5% / 20px 25em 30px 35em;<br/><br/>/* Global values */<br/>border-radius: inherit;<br/>border-radius: initial;<br/>border-radius: revert;<br/>border-radius: revert-layer;<br/>border-radius: unset;<br/>```<br/><br/>The `border-radius` property is specified as:<br/><br/>- one, two, three, or four {{cssxref(&lt;length&gt;)}} or {{cssxref(&lt;percentage&gt;)}} values. This is used to set a single radius for the corners.<br/>- followed optionally by / and one, two, three, or four `<length>` or `<percentage>` values. This is used to set an additional radius, so you can have elliptical corners.<br/><br/>### Values<br/><br/><table><br/>  <tbody><br/>    <tr><br/>      <td><em>radius</em></td><br/>      <td><img alt=all-corner.png src=all-corner.png /></td><br/>      <td><br/>        Is a {{cssxref(&lt;length&gt;)}} or a<br/>        {{cssxref(&lt;percentage&gt;)}} denoting a radius to use<br/>        for the border in each corner of the border.
