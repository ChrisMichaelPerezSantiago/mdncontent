## ✔️ appearance 
 ---<br/>title: appearance<br/>slug: Web/CSS/appearance<br/>tags:<br/>  - '-moz-appearance'<br/>  - '-webkit-appearance'<br/>  - CSS<br/>  - CSS Basic User Interface<br/>  - CSS Property<br/>  - Reference<br/>  - appearance<br/>  - recipe:css-property<br/>browser-compat: css.properties.appearance<br/>---<br/>{{CSSRef}}<br/><br/>The **`appearance`** CSS property is used to control native appearance of UI controls, that are based on operating system's theme.<br/><br/>{{EmbedInteractiveExample(pages/css/appearance.html)}}<br/><br/>Before standardization this property allowed simple elements to be shown as widgets, such as buttons or check boxes. It was considered a misfeature and authors are encouraged to use only standard keywords now.<br/><br/>> **Note:** If you wish to use this property on websites, you should test it very carefully. Although it is supported in most modern browsers, its implementation varies. In older browsers, even the keyword `none` does not have the same effect on all form elements across different browsers, and some do not support it at all. The differences are smaller in the newest browsers.<br/><br/>## Syntax<br/><br/>```css<br/>/* CSS Basic User Interface Module Level 4 values */<br/>appearance: none;<br/>appearance: auto;<br/>appearance: menulist-button;<br/>appearance: textfield;<br/><br/>/* Compat-auto values, which have the same effect as 'auto' */<br/>appearance: button;<br/>appearance: searchfield;<br/>appearance: textarea;<br/>appearance: push-button;<br/>appearance: slider-horizontal;<br/>appearance: checkbox;<br/>appearance: radio;<br/>appearance: square-button;<br/>appearance: menulist;<br/>appearance: listbox;<br/>appearance: meter;<br/>appearance: progress-bar;<br/><br/>/* Partial list of available values in Gecko */<br/>-moz-appearance: scrollbarbutton-up;<br/>-moz-appearance: button-bevel;<br/><br/>/* Partial list of available values in WebKit/Blink (as well as Gecko and Edge) */<br/>-webkit-appearance: media-mute-button;<br/>-webkit-appearance: caret;<br/><br/>/* Global values */<br/>appearance: inherit;<br/>appearance: initial;<br/>appearance: revert;<br/>appearance: revert-layer;<br/>appearance: unset;<br/>```<br/><br/>### Values<br/><br/>#### Standard keywords<br/><br/><table class=standard-table><br/>  <tbody><br/>    <tr><br/>      <th>Value</th><br/>      <th>Browser</th><br/>      <th>Description</th><br/>    </tr><br/>    <tr><br/>      <td><code>none</code></td><br/>      <td>Firefox Chrome Safari Edge</td><br/>      <td>Hides certain features of widgets, such as arrow displayed in select element, indicating that list can be expanded.</td><br/>    </tr><br/>    <tr><br/>      <td><code>auto</code></td><br/>      <td>Firefox Chrome Edge</td><br/>      <td><br/>        The user agent selects the appropriate special styling based on the<br/>        element.
