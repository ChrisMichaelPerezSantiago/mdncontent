## ✔️ font 
 ---<br/>title: font<br/>slug: Web/CSS/font<br/>tags:<br/>  - CSS<br/>  - CSS Fonts<br/>  - CSS Property<br/>  - Reference<br/>  - recipe:css-shorthand-property<br/>browser-compat: css.properties.font<br/>---<br/>{{CSSRef}}<br/><br/>The **`font`** CSS [shorthand property](/en-US/docs/Web/CSS/Shorthand_properties) sets all the different properties of an element's font. Alternatively, it sets an element's font to a system font.<br/><br/>{{EmbedInteractiveExample(pages/css/font.html)}}<br/><br/>As with any shorthand property, any individual value that is not specified is set to its corresponding initial value (possibly overriding values previously set using non-shorthand properties). Though not directly settable by `font`, the longhands {{cssxref(font-size-adjust)}} and {{cssxref(font-kerning)}} are also reset to their initial values.<br/><br/>## Constituent properties<br/><br/>This property is a shorthand for the following CSS properties:<br/><br/>- {{cssxref(font-family)}}<br/>- {{cssxref(font-size)}}<br/>- {{cssxref(font-stretch)}}<br/>- {{cssxref(font-style)}}<br/>- {{cssxref(font-variant)}}<br/>- {{cssxref(font-weight)}}<br/>- {{cssxref(line-height)}}<br/><br/>## Syntax<br/><br/>The `font` property may be specified as either a single keyword, which will select a system font, or as a shorthand for various font-related properties.<br/><br/>If `font` is specified as a system keyword, it must be one of: `caption`, `icon`, `menu`, `message-box`, `small-caption`, `status-bar`.<br/><br/>If `font` is specified as a shorthand for several font-related properties, then:<br/><br/>- it must include values for:<br/><br/>  - {{cssxref(&lt;font-size&gt;)}}<br/>  - {{cssxref(&lt;font-family&gt;)}}<br/><br/>- it may optionally include values for:<br/><br/>  - {{cssxref(&lt;font-style&gt;)}}<br/>  - {{cssxref(&lt;font-variant&gt;)}}<br/>  - {{cssxref(&lt;font-weight&gt;)}}<br/>  - {{cssxref(&lt;font-stretch&gt;)}}<br/>  - {{cssxref(&lt;line-height&gt;)}}<br/><br/>- `font-style`, `font-variant` and `font-weight` must precede `font-size`<br/>- `font-variant` may only specify the values defined in CSS 2.1, that is `normal` and `small-caps`<br/>- `font-stretch` may only be a single keyword value.<br/>- `line-height` must immediately follow `font-size`, preceded by /, like this: `16px/3`<br/>- `font-family` must be the last value specified.<br/><br/>### Values<br/><br/>- `<'font-style'>`<br/>  - : See the {{cssxref(font-style)}} CSS property.<br/>- `<'font-variant'>`<br/>  - : See the {{cssxref(font-variant)}} CSS property.<br/>- `<'font-weight'>`<br/>  - : See the {{cssxref(font-weight)}} CSS property.<br/>- `<'font-stretch'>`<br/>  - : See the {{cssxref(font-stretch)}} CSS property.<br/>- `<'font-size'>`<br/>  - : See the {{cssxref(font-size)}} CSS property.<br/>- `<'line-height'>`<br/>  - : See the {{cssxref(line-height)}} CSS property.<br/>- `<'font-family'>`<br/>  - : See the {{cssxref(font-family)}} CSS property.<br/><br/>#### System font values<br/><br/>- `caption`<br/>  - : The system font used for captioned controls (e.g., buttons, drop-downs, etc.).<br/>- `icon`<br/>  - : The system font used to label icons.<br/>- `menu`<br/>  - : The system font used in menus (e.g., dropdown menus and menu lists).<br/>- `message-box`<br/>  - : The system font used in dialog boxes.<br/>- `small-caption`<br/>  - : The system font used for labeling small controls.<br/>- `status-bar`<br/>  - : The system font used in window status bars.<br/>- Prefixed system font keywords<br/>  - : Browsers often implement several more, prefixed, keywords: Gecko implements `-moz-window`, `-moz-document`, `-moz-desktop`, `-moz-info`, `-moz-dialog`, `-moz-button`, `-moz-pull-down-menu`, `-moz-list`, and `-moz-field`.<br/><br/>## Formal definition<br/><br/>{{CSSInfo}}<br/><br/>## Formal syntax<br/><br/>{{CSSSyntax}}<br/><br/>## Examples<br/><br/>### Setting font properties<br/><br/>```css<br/>/* Set the font size to 12px and the line height to 14px.<br/>   Set the font family to sans-serif */<br/>p { font: 12px/14px sans-serif }<br/><br/>/* Set the font size to 80% of the parent element<br/>   or default value (if no parent element present).<br/>   Set the font family to sans-serif */<br/>p { font: 80% sans-serif }<br/><br/>/* Set the font weight to bold,<br/>   the font-style to italic,<br/>   the font size to large,<br/>   and the font family to serif. */<br/>p { font: bold italic large serif }<br/><br/>/* Use the same font as the status bar of the window */<br/>p { font: status-bar }<br/>```<br/><br/>###
