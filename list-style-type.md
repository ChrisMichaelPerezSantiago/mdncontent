## ✔️ list-style-type 
 ---<br/>title: list-style-type<br/>slug: Web/CSS/list-style-type<br/>tags:<br/>  - CSS<br/>  - CSS Lists<br/>  - CSS Property<br/>  - Reference<br/>  - recipe:css-property<br/>browser-compat: css.properties.list-style-type<br/>---<br/>{{CSSRef}}<br/><br/>The **`list-style-type`** [CSS](/en-US/docs/Web/CSS) property sets the marker (such as a disc, character, or custom counter style) of a list item element.<br/><br/>{{EmbedInteractiveExample(pages/css/list-style-type.html)}}<br/><br/>The [color](/en-US/docs/Web/CSS/color_value) of the marker will be the same as the computed color of the element it applies to.<br/><br/>Only a few elements ({{HTMLElement(li)}} and {{HTMLElement(summary)}}) have a default value of `display: list-item`. However, the `list-style-type` property may be applied to any element whose {{cssxref(display)}} value is set to `list-item`. Moreover, because this property is inherited, it can be set on a parent element (commonly {{HTMLElement(ol)}} or {{HTMLElement(ul)}}) to make it apply to all list items.<br/><br/>## Syntax<br/><br/>```css<br/>/* Partial list of types */<br/>list-style-type: disc;<br/>list-style-type: circle;<br/>list-style-type: square;<br/>list-style-type: decimal;<br/>list-style-type: georgian;<br/>list-style-type: trad-chinese-informal;<br/>list-style-type: kannada;<br/><br/>/* <string> value */<br/>list-style-type: '-';<br/><br/>/* Identifier matching an @counter-style rule */<br/>list-style-type: custom-counter-style;<br/><br/>/* Keyword value */<br/>list-style-type: none;<br/><br/>/* Global values */<br/>list-style-type: inherit;<br/>list-style-type: initial;<br/>list-style-type: revert;<br/>list-style-type: revert-layer;<br/>list-style-type: unset;<br/>```<br/><br/>The `list-style-type` property may be defined as any one of:<br/><br/>- a `<custom-ident>` value<br/>- a `symbols()` value<br/>- a `<string>` value<br/>- the keyword `none`.<br/><br/>Note that:<br/><br/>- Some types require a suitable font installed to display as expected.<br/>- The `cjk-ideographic` is identical to `trad-chinese-informal`; it exists for legacy reasons.<br/><br/>### Values<br/><br/>- {{cssxref(custom-ident, &lt;custom-ident&gt;)}}<br/>  - : A identifier matching the value of a {{cssxref(@counter-style)}} or one of the predefined styles:<br/>- {{cssxref(symbols, symbols())}}<br/>  - : Defines an anonymous style of the list.<br/>- {{cssxref(&lt;string&gt;)}}<br/>  - : The specified string will be used as the item's marker.<br/>- `none`<br/>  - : No item marker is shown.<br/>- `disc`<br/>  - : A filled circle (default value).<br/>- `circle`<br/>  - : A hollow circle.<br/>- `square`<br/>  - : A filled square.<br/>- `decimal`<br/>  - : Decimal numbers, beginning with 1.<br/>- `cjk-decimal` {{experimental_inline}}<br/>  - : Han decimal numbers.<br/>- `decimal-leading-zero`<br/>  - : Decimal numbers, padded by initial zeros.<br/>- `lower-roman`<br/>  - : Lowercase roman numerals.<br/>- `upper-roman`<br/>  - : Uppercase roman numerals.<br/>- `lower-greek`<br/>  - : Lowercase classical Greek.<br/>- `lower-alpha`, `lower-latin`<br/>  - : Lowercase ASCII letters.<br/>- `upper-alpha`, `upper-latin`<br/>  - : Uppercase ASCII letters.<br/>- `arabic-indic`, `-moz-arabic-indic`<br/>  - : Arabic-Indic numbers.<br/>- `armenian`<br/>  - : Traditional Armenian numbering.<br/>- `bengali`, `-moz-bengali`<br/>  - : Bengali numbering.<br/>- `cambodian`/`khmer`<br/>  - : Cambodian/Khmer numbering.<br/>- `cjk-earthly-branch`, `-moz-cjk-earthly-branch`<br/>  - : Han Earthly Branch ordinals.<br/>- `cjk-heavenly-stem`, `-moz-cjk-heavenly-stem`<br/>  - : Han Heavenly Stem ordinals.<br/>- `cjk-ideographic`{{experimental_inline}}<br/>  - : Identical to `trad-chinese-informal`.<br/>- `devanagari`, `-moz-devanagari`<br/>  - : Devanagari numbering.<br/>- `ethiopic-numeric` {{experimental_inline}}<br/>  - : Ethiopic numbering.<br/>- `georgian`<br/>  - : Traditional Georgian numbering.<br/>- `gujarati`, `-moz-gujarati`<br/>  - : Gujarati numbering.<br/>- `gurmukhi`, `-moz-gurmukhi`<br/>  - : Gurmukhi numbering.<br/>- `hebrew` {{experimental_inline}}<br/>  - : Traditional Hebrew numbering<br/>- `hiragana` {{experimental_inline}}<br/>  - : Dictionary-order hiragana lettering.<br/>- `hiragana-iroha` {{experimental_inline}}<br/>  - : {{interwiki('wikipedia', 'Iroha', 'Iroha-order')}} hiragana lettering<br/>- `japanese-formal` {{experimental_inline}}<br/>  - : Japanese formal numbering to be used in legal or financial documents. The kanjis are designed so that they can't be modified to look like another correct one.<br/>- `japanese-informal` {{experimental_inline}}<br/>  - : Japanese informal numbering.<br/>- `kannada`, `-moz-kannada`<br/>  - : Kannada numbering.<br/>- `katakana` {{experimental_inline}}<br/>  - : Dictionary-order katakana lettering<br/>- `katakana-iroha` {{experimental_inline}}<br/>  - : {{interwiki('wikipedia', 'Iroha', 'Iroha-order')}} katakana lettering<br/>- `korean-hangul-formal` {{experimental_inline}}<br/>  - : Korean hangul numbering.<br/>- `korean-hanja-formal` {{experimental_inline}}<br/>  - : Formal Korean Han numbering.<br/>- `korean-hanja-informal` {{experimental_inline}}<br/>  - : Korean hanja numbering.<br/>- `lao`, `-moz-lao`<br/>  - : Laotian numbering.<br/>- `lower-armenian` {{experimental_inline}}\*<br/>  - : Lowercase Armenian numbering.<br/>- `malayalam`, `-moz-malayalam`<br/>  - : Malayalam numbering.<br/>- `mongolian` {{experimental_inline}}<br/>  - : Mongolian numbering.<br/>- `myanmar`, `-moz-myanmar`<br/>  - : Myanmar (Burmese) numbering.<br/>- `oriya`, `-moz-oriya`<br/>  - : Oriya numbering.<br/>- `persian` {{experimental_inline}}, `-moz-persian`<br/>  - : Persian numbering<br/>- `simp-chinese-formal` {{experimental_inline}}<br/>  - : Simplified Chinese formal numbering.<br/>- `simp-chinese-informal` {{experimental_inline}}<br/>  - : Simplified Chinese informal numbering.<br/>- `tamil` {{experimental_inline}}, `-moz-tamil`<br/>  - : Tamil numbering.<br/>- `telugu`, `-moz-telugu`<br/>  - : Telugu numbering.<br/>- `thai`, `-moz-thai`<br/>  - : Thai numbering.<br/>- `tibetan` {{experimental_inline}}\*<br/>  - : Tibetan numbering.<br/>- `trad-chinese-formal` {{experimental_inline}}<br/>  - : Traditional Chinese formal numbering.<br/>- `trad-chinese-informal` {{experimental_inline}}<br/>  - : Traditional Chinese informal numbering.<br/>- `upper-armenian` {{experimental_inline}}\*<br/>  - : Traditional uppercase Armenian numbering.<br/>- `disclosure-open` {{experimental_inline}}<br/>  - : Symbol indicating that a disclosure widget such as {{HTMLElement(details)}} is opened.<br/>- `disclosure-closed` {{experimental_inline}}<br/>  - : Symbol indicating that a disclosure widget, like {{HTMLElement(details)}} is closed.<br/><br/>### Non-standard extensions<br/><br/>A few more predefined types are provided by Mozilla (Firefox), Blink (Chrome and Opera) and WebKit (Safari) to support list types in other languages. See the compatibility table to check which browsers supports which extension.<br/><br/>- `-moz-ethiopic-halehame`<br/>- `-moz-ethiopic-halehame-am`<br/>- `ethiopic-halehame-ti-er`, `-moz-ethiopic-halehame-ti-er`<br/>- `ethiopic-halehame-ti-et`, `-moz-ethiopic-halehame-ti-et`<br/>- `hangul`, `-moz-hangul`<br/>- `hangul-consonant`, `-moz-hangul-consonant`<br/>- `urdu`, `-moz-urdu`<br/><br/>## Accessibility concerns<br/><br/>The [VoiceOver](https://help.apple.com/voiceover/info/guide/) screen reader has an issue where unordered lists with a `list-style-type` value of `none` applied to them will not be announced as a list. To address this, add a [zero-width space](https://en.wikipedia.org/wiki/Zero-width_space) as [pseudo content](/en-US/docs/Web/CSS/content) before each list item to ensure the list is announced properly.
