## ✔️ font-weight 
 ---<br/>title: font-weight<br/>slug: Web/CSS/font-weight<br/>tags:<br/>  - CSS<br/>  - CSS Fonts<br/>  - CSS Property<br/>  - Reference<br/>  - recipe:css-property<br/>browser-compat: css.properties.font-weight<br/>---<br/>{{CSSRef}}<br/><br/>The **`font-weight`** [CSS](/en-US/docs/Web/CSS) property sets the weight (or boldness) of the font. The weights available depend on the {{cssxref(font-family)}} that is currently set.<br/><br/>{{EmbedInteractiveExample(pages/css/font-weight.html)}}<br/><br/>## Syntax<br/><br/>```css<br/>/* Keyword values */<br/>font-weight: normal;<br/>font-weight: bold;<br/><br/>/* Keyword values relative to the parent */<br/>font-weight: lighter;<br/>font-weight: bolder;<br/><br/>/* Numeric keyword values */<br/>font-weight: 100;<br/>font-weight: 200;<br/>font-weight: 300;<br/>font-weight: 400;// normal<br/>font-weight: 500;<br/>font-weight: 600;<br/>font-weight: 700;// bold<br/>font-weight: 800;<br/>font-weight: 900;<br/><br/>/* Global values */<br/>font-weight: inherit;<br/>font-weight: initial;<br/>font-weight: revert;<br/>font-weight: revert-layer;<br/>font-weight: unset;<br/>```<br/><br/>The `font-weight` property is specified using any one of the values listed below.<br/><br/>### Values<br/><br/>- `normal`<br/>  - : Normal font weight. Same as `400`.<br/>- `bold`<br/>  - : Bold font weight. Same as `700`.<br/>- `lighter`<br/>  - : One relative font weight lighter than the parent element. Note that only four font weights are considered for relative weight calculation; see the [Meaning of relative weights](#meaning_of_relative_weights) section below.<br/>- `bolder`<br/>  - : One relative font weight heavier than the parent element. Note that only four font weights are considered for relative weight calculation; see the [Meaning of relative weights](#meaning_of_relative_weights) section below.<br/>- `<number>`<br/>  - : A {{cssxref(&lt;number&gt;)}} value between 1 and 1000, inclusive. Higher numbers represent weights that are bolder than (or as bold as) lower numbers. Certain commonly used values correspond to common weight names, as described in the [Common weight name mapping](#common_weight_name_mapping) section below.<br/><br/>In earlier versions of the `font-weight` specification, the property accepts only keyword values and the numeric values 100, 200, 300, 400, 500, 600, 700, 800, and 900; non-variable fonts can only really make use of these set values, although fine-grained values (e.g. 451) will be translated to one of these values for non-variable fonts using the [Fallback weights](#fallback_weights) system.<br/><br/>CSS Fonts Level 4 extends the syntax to accept any number between 1 and 1000 and introduces [Variable fonts](#variable_fonts), which can make use of this much finer-grained range of font weights.<br/><br/>### Fallback weights<br/><br/>If the exact weight given is unavailable, then the following rule is used to determine the weight actually rendered:<br/><br/>- If the target weight given is between `400` and `500` inclusive:<br/><br/>  - Look for available weights between the target and `500`, in ascending order.<br/>  - If no match is found, look for available weights less than the target, in descending order.<br/>  - If no match is found, look for available weights greater than `500`, in ascending order.<br/><br/>- If a weight less than `400` is given, look for available weights less than the target, in descending order. If no match is found, look for available weights greater than the target, in ascending order.<br/>- If a weight greater than `500` is given, look for available weights greater than the target, in ascending order. If no match is found, look for available weights less than the target, in descending order.<br/><br/>### Meaning of relative weights<br/><br/>When `lighter` or `bolder` is specified, the below chart shows how the absolute font weight of the element is determined.<br/><br/>Note that when using relative weights, only four font weights are considered — thin (100), normal (400), bold (700), and heavy (900). one of the numbers in [Common weight name mapping](#common_weight_name_mapping). experiencing low vision conditions may have difficulty reading text set with a `font-weight` value of `100` (Thin/Hairline) or `200` (Extra Light), especially if the font has a [low contrast color ratio](/en-US/docs/Web/CSS/color#accessibility_concerns).<br/><br/>- [MDN Understanding WCAG, Guideline 1.4
