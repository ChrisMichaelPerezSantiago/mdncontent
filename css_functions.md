 ---<br/>title: CSS Functional Notation<br/>slug: Web/CSS/CSS_Functions<br/>tags:<br/>  - CSS<br/>  - CSS Data Type<br/>  - CSS Function<br/>  - Functional Notation<br/>  - Functions<br/>  - Reference<br/>  - Types<br/>  - data types<br/>spec-urls:<br/>  - https://drafts.csswg.org/css-values/<br/>  - https://drafts.csswg.org/css-color/<br/>  - https://drafts.csswg.org/css-images/<br/>---<br/>{{CSSRef}}<br/><br/>**CSS functional notation** is a type of [CSS value](/en-US/docs/Web/CSS/CSS_Values_and_Units) that can represent more complex data types or invoke special data processing or calculations.<br/><br/>## Syntax<br/><br/>```css<br/>selector {<br/>  property: functional-notation( [argument]? [, argument]! );<br/>}<br/>```<br/><br/>The syntax starts with the **name of the functional notation**, followed by a left parenthesis `(`. Next up are the notation argument(s), and the function is finished off with a closing parenthesis `)`.<br/><br/>Functions can take multiple arguments, which are formatted similarly to CSS property values. Whitespace is allowed, but they are optional inside the parentheses. In some functional notations multiple arguments are separated by commas, while others use spaces.<br/><br/>## Transform functions<br/><br/>These functions are used when the {{cssxref(&lt;transform-function&gt;)}} CSS [data type](/en-US/docs/Web/CSS/CSS_Types) is used as a value of {{cssxref(transform)}}.<br/><br/>- {{cssxref(transform-function/matrix, matrix())}}<br/>  - : Describes a homogeneous 2D transformation matrix.<br/>- {{cssxref(transform-function/matrix3d, matrix3d())}}<br/>  - : Describes a 3D transformation as a 4×4 homogeneous matrix.<br/>- {{cssxref(transform-function/perspective, perspective())}}<br/>  - : Sets the distance between the user and the z=0 plane.<br/>- {{cssxref(transform-function/rotate, rotate())}}<br/>  - : Rotates an element around a fixed point on the 2D plane.<br/>- {{cssxref(transform-function/rotate3d, rotate3d())}}<br/>  - : Rotates an element around a fixed axis in 3D space.<br/>- {{cssxref(transform-function/rotateX, rotateX())}}<br/>  - : Rotates an element around the horizontal axis.<br/>- {{cssxref(transform-function/rotateY, rotateY())}}<br/>  - : Rotates an element around the vertical axis.<br/>- {{cssxref(transform-function/rotateZ, rotateZ())}}<br/>  - : Rotates an element around the z-axis.<br/>- {{cssxref(transform-function/scale, scale())}}<br/>  - : Scales an element up or down on the 2D plane.<br/>- {{cssxref(transform-function/scale3d, scale3d())}}<br/>  - : Scales an element up or down in 3D space.<br/>- {{cssxref(transform-function/scaleX, scaleX())}}<br/>  - : Scales an element up or down horizontally.<br/>- {{cssxref(transform-function/scaleY, scaleY())}}<br/>  - : Scales an element up or down vertically.<br/>- {{cssxref(transform-function/scaleZ, scaleZ())}}<br/>  - : Scales an element up or down along the z-axis.<br/>- {{cssxref(transform-function/skew, skew())}}<br/>  - : Skews an element on the 2D plane.<br/>- {{cssxref(transform-function/skewX, skewX())}}<br/>  - : Skews an element in the horizontal direction.<br/>- {{cssxref(transform-function/skewY, skewY())}}<br/>  - : Skews an element in the vertical direction.<br/>- {{cssxref(transform-function/translate, translate())}}<br/>  - : Translates an element on the 2D plane.<br/>- {{cssxref(transform-function/translate3d, translate3d())}}<br/>  - : Translates an element in 3D space.<br/>- {{cssxref(transform-function/translateX, translateX())}}<br/>  - : Translates an element horizontally.<br/>- {{cssxref(transform-function/translateY, translateY())}}<br/>  - : Translates an element vertically.<br/>- {{cssxref(transform-function/translateZ, translateZ())}}<br/>  - : Translates an element along the z-axis.<br/><br/>## Math functions<br/><br/>The math functions allow CSS numeric values to be written as mathematical expressions.<br/><br/>- {{cssxref(calc, calc())}}<br/>  - : A math function that allows basic arithmetic to be performed on numerical CSS values.<br/>- {{cssxref(clamp, clamp())}}<br/>  - : A comparison function that takes a minimum, central, and maximum value and represents its central calculation.<br/>- {{cssxref(max, max())}}<br/>  - : A comparison function that represents the largest of a list of values.<br/>- {{cssxref(min, min())}}<br/>  - : A comparison function that represents the smallest of a list of values.<br/>- {{cssxref(abs, abs())}} {{Experimental_Inline}}<br/>  - : Takes a calculation and returns the absolute value.<br/>- {{cssxref(acos, acos())}} {{Experimental_Inline}}<br/>  - : An inverse trigonometric function the angle returned must be normalized to the range \[0deg, 180deg].<br/>- {{cssxref(asin, asin())}} {{Experimental_Inline}}<br/>  - : An inverse trigonometric function the angle returned must be normalized to the range \[-90deg, 90deg].<br/>- {{cssxref(atan, atan())}} {{Experimental_Inline}}<br/>  - : An inverse trigonometric function the angle returned must be normalized to the range \[-90deg, 90deg].<br/>- {{cssxref(atan2, atan2())}} {{Experimental_Inline}}<br/>  - : Contains two comma-separated calculations, A and B. A and B can resolve to any {{cssxref(&lt;number&gt;)}}, {{cssxref(&lt;dimension&gt;)}}, or {{cssxref(&lt;percentage&gt;)}}, but must have the same type, or else the function is invalid.<br/>- {{cssxref(cos, cos())}} {{Experimental_Inline}}<br/>  - : Contains a single calculation which must resolve to either a {{cssxref(&lt;number&gt;)}} or an {{cssxref(&lt;angle&gt;)}}.<br/>- {{cssxref(exp, exp())}} {{Experimental_Inline}}<br/>  - : Contains one calculation which must resolve to a {{cssxref(&lt;number&gt;)}}, and returns the same value as pow(e, A) as a {{cssxref(&lt;number&gt;)}}.<br/>- {{cssxref(hypot, hypot())}} {{Experimental_Inline}}<br/>  - : Contains one or more comma-separated calculations, and returns the length of an N-dimensional vector with components equal to each of the calculations.<br/>- {{cssxref(log, log())}} {{Experimental_Inline}}<br/>  - : Contains one or two calculations (representing the value to be logarithmed, and the base of the logarithm, defaulting to e), which must both resolve as a {{cssxref(&lt;number&gt;)}}, and returns the logarithm base B of the value A, as a {{cssxref(&lt;number&gt;)}}.<br/>- {{cssxref(mod, mod())}} {{Experimental_Inline}}<br/>  - : A modulus function that contains two calculations A and B, and returns the difference between A and the nearest integer multiple of B either above or below A.<br/>- {{cssxref(pow, pow())}} {{Experimental_Inline}}<br/>  - : Contains two comma-separated calculations A and B, both of which must resolve as a {{cssxref(&lt;number&gt;)}}, and returns the result of raising A to the power of B, returning the value as a {{cssxref(&lt;number&gt;)}}.<br/>- {{cssxref(rem, rem())}} {{Experimental_Inline}}<br/>  - : A modulus function that contains two calculations A and B, and returns the difference between A and the nearest integer multiple of B either above or below A.<br/>- {{cssxref(round, round())}} {{Experimental_Inline}}<br/>  - : Contains an optional rounding strategy, and two calculations A and B, and returns the value of A, rounded according to the rounding strategy, to the nearest integer multiple of B either above or below A.<br/>- {{cssxref(sign, sign())}} {{Experimental_Inline}}<br/>  - : Takes a calculation and returns -1 if the numeric value is negative,<br/>    \+1 if the numeric value is positive, 0⁺ if the numeric value is 0⁺, and 0⁻ if the numeric value is 0⁻.<br/>- {{cssxref(sin, sin())}} {{Experimental_Inline}}<br/>  - : Contains a single calculation which must resolve to either a {{cssxref(&lt;number&gt;)}} or an {{cssxref(&lt;angle&gt;)}}.<br/>- {{cssxref(sqrt, sqrt())}} {{Experimental_Inline}}<br/>  - : Contains a single calculation which must resolve to a {{cssxref(&lt;number&gt;)}}, and returns the square root of the value as a {{cssxref(&lt;number&gt;)}}.<br/>- {{cssxref(tan, tan())}} {{Experimental_Inline}}<br/>  - : Contains a single calculation which must resolve to either a {{cssxref(&lt;number&gt;)}} or an {{cssxref(&lt;angle&gt;)}}.<br/><br/>## Filter functions<br/><br/>The **`<filter-function>`** [CSS](/en-US/docs/Web/CSS) [data type](/en-US/docs/Web/CSS/CSS_Types) represents a graphical effect that can change the appearance of an input image. It is used in the {{cssxref(filter)}} and {{cssxref(backdrop-filter)}} properties.<br/><br/>- {{cssxref(filter-function/blur, blur())}}<br/>  - : Blurs the image.<br/>- {{cssxref(filter-function/brightness, brightness())}}<br/>  - : Makes the image brighter or darker.<br/>- {{cssxref(filter-function/contrast, contrast())}}<br/>  - : Increases or decreases the image's contrast.<br/>- {{cssxref(filter-function/drop-shadow, drop-shadow())}}<br/>  - : Applies a drop shadow behind the image.<br/>- {{cssxref(filter-function/grayscale, grayscale())}}<br/>  - : Converts the image to grayscale.<br/>- {{cssxref(filter-function/hue-rotate, hue-rotate())}}<br/>  - : Changes the overall hue of the image.<br/>- {{cssxref(filter-function/invert, invert())}}<br/>  - : Inverts the colors of the image.<br/>- {{cssxref(filter-function/opacity, opacity())}}<br/>  - : Makes the image transparent.<br/>- {{cssxref(filter-function/saturate, saturate())}}<br/>  - : Super-saturates or desaturates the input image.<br/>- {{cssxref(filter-function/sepia, sepia())}}<br/>  - : Converts the image to sepia.<br/><br/>## Color functions<br/><br/>Functions which specify different color representations. These may be used anywhere a {{cssxref(&lt;color_value&gt;,&lt;color&gt;)}} is valid.<br/><br/>- {{cssxref(color_value/color, color())}} {{Experimental_Inline}}<br/>  - : Allows a color to be specified in a particular, specified colorspace (rather than the implicit sRGB colorspace that most of the other color functions operate in).<br/>- {{cssxref(color_value/color-mix, color-mix())}} {{Experimental_Inline}}<br/>  - : Takes two {{cssxref(color_value,color)}} values and returns the result of mixing them in a given colorspace by a given amount.<br/>- {{cssxref(color_value/color-contrast, color-contrast())}} {{Experimental_Inline}}<br/>  - : Takes a {{cssxref(color_value,color)}} value and compares it to a list of other {{cssxref(color_value,color)}} values, selecting the one with the highest contrast from the list.<br/>- {{cssxref(color_value/device-cmyk, device-cmyk())}} {{Experimental_Inline}}<br/>  - : Used to express CMYK colors in a device-independent way.<br/>- {{cssxref(color_value/hsl, hsl())}}<br/>  - : The HSL color model defines a given color according to its hue, saturation, and lightness components. An optional alpha component represents the color's transparency.<br/>- {{cssxref(color_value/hsla, hsla())}}<br/>  - : The HSL color model defines a given color according to its hue, saturation, and lightness components. The alpha component represents the color's transparency.<br/>- {{cssxref(color_value/hwb, hwb())}} {{Experimental_Inline}}<br/>  - : HWB (short for Hue-Whiteness-Blackness) is another method of specifying colors, similar to HSL.<br/>- {{cssxref(color_value/lab, lab())}} {{Experimental_Inline}}<br/>  - : Lab color is device-independent and specifies physical measurements of color.<br/>- {{cssxref(color_value/lch, lch())}} {{Experimental_Inline}}<br/>  - : LCH color is device-independent and specifies color using polar coordinates for chroma and hue.<br/>- {{cssxref(color_value/rgb, rgb())}}<br/>  - : The RGB color model defines a given color according to its red, green, and blue components. An optional alpha component represents the color's transparency.<br/>- {{cssxref(color_value/rgba, rgba())}}<br/>  - : The RGB color model defines a given color according to its red, green, and blue components. The alpha component represents the color's transparency.<br/><br/>## Image functions<br/><br/>These functions may be used wherever an {{cssxref(&lt;image&gt;)}} is valid as the value for a property.<br/><br/>- {{cssxref(gradient/conic-gradient, conic-gradient())}}<br/>  - : Conic gradients transition colors progressively around a circle.<br/>- {{cssxref(image/image,image())}} {{Experimental_Inline}}<br/>  - : Defines an {{cssxref(&lt;image&gt;)}} in a similar fashion to the {{cssxref(url, url())}} function, but with added functionality including specifying the image's directionality and fallback images for when the preferred image is not supported.<br/>- {{cssxref(image/image-set,image-set())}} {{Experimental_Inline}}<br/>  - : A method of letting the browser pick the most appropriate CSS image from a given set, primarily for high pixel density screens.<br/>- {{cssxref(gradient/linear-gradient,linear-gradient())}}<br/>  - : Linear gradients transition colors progressively along an imaginary line.<br/>- {{cssxref(gradient/radial-gradient,radial-gradient())}}<br/>  - : Radial gradients transition colors progressively from a center point (origin).<br/>- {{cssxref(gradient/repeating-linear-gradient,repeating-linear-gradient())}}<br/>  - : Is similar to `linear-gradient()` and takes the same arguments, but it repeats the color stops infinitely in all directions so as to cover its entire container.<br/>- {{cssxref(gradient/repeating-radial-gradient,repeating-radial-gradient())}}<br/>  - : Is similar to `radial-gradient()` and takes the same arguments, but it repeats the color stops infinitely in all directions so as to cover its entire container.<br/>- {{cssxref(gradient/repeating-conic-gradient,repeating-conic-gradient())}}<br/>  - : Is similar to `conic-gradient()` and takes the same arguments, but it repeats the color stops infinitely in all directions so as to cover its entire container.<br/>- {{cssxref(cross-fade, cross-fade())}}<br/>  - : Can be used to blend two or more images at a defined transparency.<br/>- {{cssxref(element, element())}}<br/>  - : Defines an `<image>` value generated from an arbitrary HTML element.<br/>- {{cssxref(paint, paint())}}<br/>  - : Defines an `<image>` value generated with a PaintWorklet.<br/><br/>## Counter functions<br/><br/>The counter functions are generally used with the {{cssxref(content)}} property, although in theory may be used wherever a {{cssxref(&lt;string&gt;)}} is supported.<br/><br/>- {{cssxref(counter, counter())}}<br/>  - : Returns a string representing the current value of the named counter, if there is one.<br/>- {{cssxref(counters, counters())}}<br/>  - : Enables nested counters, returning a concatenated string representing the current values of the named counters, if there are any.<br/>- {{cssxref(symbols, symbols())}}<br/>  - : Lets you define counter styles inline, directly as the value of a property.<br/><br/>## Font functions<br/><br/>The {{cssxref(font-variant-alternates)}} property controls the use of alternate glyphs, the following functions are values for this property.<br/><br/>- {{cssxref(font-variant-alternates/stylistic, stylistic())}}<br/>  - : This function enables stylistic alternates for individual characters. The parameter is a font-specific name mapped to a number. It corresponds to the OpenType value `salt`, like `salt 2`.<br/>- {{cssxref(font-variant-alternates/styleset, styleset())}}<br/>  - : This function enables stylistic alternatives for sets of characters. The parameter is a font-specific name mapped to a number. It corresponds to the OpenType value `ssXY`, like `ss02`.<br/>- {{cssxref(font-variant-alternates/character-variant, character-variant())}}<br/>  - : This function enables specific stylistic alternatives for characters. It is similar to `styleset()`, but doesn't create coherent glyphs for a set of characters; individual characters will have independent and not necessarily coherent styles. The parameter is a font-specific name mapped to a number. It corresponds to the OpenType value `cvXY`, like `cv02`.<br/>- {{cssxref(font-variant-alternates/swash, swash())}}<br/>  - : This function enables [swash](https://en.wikipedia.org/wiki/Swash_%28typography%29) glyphs. The parameter is a font-specific name mapped to a number. It corresponds to the OpenType values `swsh` and `cswh`, like `swsh 2` and `cswh 2`.<br/>- {{cssxref(font-variant-alternates/ornaments, ornaments())}}<br/>  - : This function enables ornaments, like [fleurons](https://en.wikipedia.org/wiki/Fleuron_%28typography%29) and other dingbat glyphs. The parameter is a font-specific name mapped to a number. It corresponds to the OpenType value `ornm`, like `ornm 2`.<br/>- {{cssxref(font-variant-alternates/annotation, annotation())}}<br/>  - : This function enables annotations, like circled digits or inverted characters. The parameter is a font-specific name mapped to a number. It corresponds to the OpenType value `nalt`, like `nalt 2`.<br/><br/>## Shape functions<br/><br/>The following functions may be used as values for the {{cssxref(&lt;basic-shape&gt;)}} data type, which is used in the {{cssxref(clip-path)}}, {{cssxref(offset-path)}}, and {{cssxref(shape-outside)}} properties.<br/><br/>- {{cssxref(basic-shape/circle,circle())}}<br/>  - : Defines a circle.<br/>- {{cssxref(basic-shape/ellipse,ellipse())}}<br/>  - : Defines an ellipse.<br/>- {{cssxref(basic-shape/inset,inset())}}<br/>  - : Defines an inset rectangle.<br/>- {{cssxref(basic-shape/polygon,polygon())}}<br/>  - : Defines a polygon.<br/>- {{cssxref(path, path())}}<br/>  - : Accepts an SVG path string to enable a shape to be drawn.<br/><br/>## Reference functions<br/><br/>The following functions are used as a value of properties to reference a value defined elsewhere.<br/><br/>- {{cssxref(attr, attr())}}<br/>  - : Used to retrieve the value of an attribute of the selected element and use it in the stylesheet.<br/>- {{cssxref(env, env())}}<br/>  - : Used to insert the value of a user agent-defined environment variable.<br/>- {{cssxref(url, url())}}<br/>  - : Used to include a file.<br/>- {{cssxref(var, var())}}<br/>  - : Used to insert a value of a custom property instead of any part of a value of another property.<br/><br/>## CSS grid functions<br/><br/>The following functions are used to define a CSS grid.<br/><br/>- {{cssxref(fit-content, fit-content())}}<br/>  - : Clamps a given size to an available size according to the formula `min(maximum size, max(minimum size, argument))`.<br/>- {{cssxref(minmax, minmax())}}<br/>  - : Defines a size range greater than or equal to _min_ and less than or equal to _max_.<br/>- {{cssxref(repeat, repeat())}}<br/>  - : Represents a repeated fragment of the track list, allowing a large number of columns or rows that exhibit a recurring pattern.<br/><br/>## Specifications<br/><br/>{{Specifications}}<br/><br/>## See also<br/><br/>- [CSS Values and Units](/en-US/docs/Web/CSS/CSS_Values_and_Units)<br/>- [Introduction to CSS: Values and Units](/en-US/docs/Learn/CSS/Building_blocks/Values_and_units)<br/>
