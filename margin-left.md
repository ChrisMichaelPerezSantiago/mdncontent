## ✔️ margin-left 
 ---<br/>title: margin-left<br/>slug: Web/CSS/margin-left<br/>tags:<br/>  - CSS<br/>  - CSS Property<br/>  - Layout<br/>  - Reference<br/>  - recipe:css-property<br/>browser-compat: css.properties.margin-left<br/>---<br/>{{CSSRef}}<br/><br/>The **`margin-left`** [CSS](/en-US/docs/Web/CSS) property sets the [margin area](/en-US/docs/Web/CSS/CSS_Box_Model/Introduction_to_the_CSS_box_model#margin_area) on the left side of an element. A positive value places it farther from its neighbors, while a negative value places it closer.<br/><br/>{{EmbedInteractiveExample(pages/css/margin-left.html)}}<br/><br/>The vertical margins of two adjacent boxes may fuse. This is called [_margin collapsing_](/en-US/docs/Web/CSS/CSS_Box_Model/Mastering_margin_collapsing).<br/><br/>In the rare cases where width is overconstrained (i.e., when all of `width`, `margin-left`, `border`, `padding`, the content area, and `margin-right` are defined), `margin-left` is ignored, and will have the same calculated value as if the `auto` value was specified.<br/><br/>## Syntax<br/><br/>```css<br/>/* <length> values */<br/>margin-left: 10px;  /* An absolute length */<br/>margin-left: 1em;   /* relative to the text size */<br/>margin-left: 5%;    /* relative to the nearest block container's width */<br/><br/>/* Keyword values */<br/>margin-left: auto;<br/><br/>/* Global values */<br/>margin-left: inherit;<br/>margin-left: initial;<br/>margin-left: revert;<br/>margin-left: revert-layer;<br/>margin-left: unset;<br/>```<br/><br/>The `margin-left` property is specified as the keyword `auto`, or a `<length>`, or a `<percentage>`. Its value can be positive, zero, or negative.<br/><br/>### Values<br/><br/>- {{cssxref(&lt;length&gt;)}}<br/>  - : The size of the margin as a fixed value.<br/>- {{cssxref(&lt;percentage&gt;)}}<br/>  - : The size of the margin as a percentage, relative to the inline size (_width_ in a horizontal language, defined by {{cssxref(writing-mode)}}) of the [containing block](/en-US/docs/Web/CSS/Containing_block).<br/>- `auto`<br/><br/>  - : The left margin receives a share of the unused horizontal space, as determined mainly by the layout mode that is used. If the values of `margin-left` and `margin-right` are both `auto`, the calculated space is evenly distributed.
